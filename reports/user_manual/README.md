# Aussie EcoLens User Manual

This folder contains the LaTeX source for the product, installation, testing, and PDF-build manual.

Build with Tectonic:

```bash
tectonic reports/user_manual/main.tex
cp reports/user_manual/main.pdf reports/Aussie_EcoLens_User_Manual.pdf
```

The generated PDF is intended to travel with the project zip so another person can install, test, and understand the project without relying on local notes.
